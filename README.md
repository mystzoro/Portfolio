## 🚀 Overview

Welcome to the source code of my personal portfolio website! This project is a highly interactive, cinematic, and professional showcase of my work as an **AI Engineer and Full Stack Developer**.

It is designed with a premium aesthetic inspired by Apple, Linear, and Stripe, focusing on **performance, smooth interactions, and 3D storytelling**.

## ✨ Key Features

- **Cinematic Hero Section:** Features interactive particle fields, orbital rings, glitch effects, magnetic buttons, and a rotating dynamic title carousel.
- **Glassmorphism UI:** Premium styling utilizing deep dark modes, glass-like cards, and subtle micro-animations.
- **Interactive Coding Desk:** A fully functional mock terminal and code editor pane that simulates neural network training and accepts Unix-like commands (e.g., `help`, `about`, `contact`).
- **Smooth Scrolling:** Integrated with Lenis for a fluid, inertial scrolling experience.
- **Responsive Design:** Completely optimized for both desktop and mobile viewports.
- **Integrated Email Delivery:** A contact form powered by Web3Forms for seamless message delivery.
- **Dynamic Projects & Skills:** 3D-tilting project cards and a detailed skills matrix highlighting proficiencies.

## 🛠️ Technology Stack

- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion, GSAP
- **Scroll Engine:** Lenis Smooth Scroll
- **Icons:** Lucide React
- **Email API:** Web3Forms

## 🏃‍♂️ Running Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/mystzoro/Portfolio.git
   cd Portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set up Web3Forms (Optional for Contact Form):**
   - Get an access key from [Web3Forms](https://web3forms.com/)
   - Replace `YOUR_WEB3FORMS_ACCESS_KEY` in `src/components/Contact.tsx`

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. **Open your browser:**
   Navigate to `http://localhost:3000`
